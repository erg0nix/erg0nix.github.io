2025-01-15

so I finally got around to setting up this site. been thinking about personal websites and the indie web lately.

why a personal website?

social media platforms come and go, and some stay way longer than you wanted and all your friends and family is on there, so you cannot just drop it, but you really want to. your content gets buried in algorithms, ads, and noise. but a personal site is yours. you control how it looks, what you write or post.

I haven’t posted anything on social media in 7 years, but I’m excited to maybe post something here.

doesn’t have to be perfect or polished. this site is pretty basic and that’s fine. it’s a place to put stuff and write about code or tools I find useful.

the indie web gets this. instead of dumping everything into platforms owned by others, you can have your own space.

might write about golang patterns, useful cli tools, or whatever I’m working on. no grand plans, just writing things down as they come up.

setup

currently Im just hosting this github with hugo, and using some actions to publish. Using a custom theme, a lot vibed since I don’t enjoy frontend that much, even though I have some strong opionions about the look and feel, but don’t like fiddling around with it too much.

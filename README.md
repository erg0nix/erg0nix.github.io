# ergonix.dev

Personal website built with Hugo and deployed to GitHub Pages.

## Development

```bash
# Install Hugo (if not already installed)
# See: https://gohugo.io/installation/

# Clone repository
git clone https://github.com/erg0nix/erg0nix.github.io.git
cd erg0nix.github.io

# Run development server
hugo server

# Build for production
hugo
```

## Deployment

Site automatically deploys to https://ergonix.dev via GitHub Actions when pushing to `main` branch.

## Tech Stack

- **Hugo** - Static site generator
- **GitHub Pages** - Hosting
- **Cloudflare** - DNS and CDN
- **Custom theme** - Terminal-inspired design with neon snow background

## Features

- Responsive design
- Custom 404 page
- RSS feed
- Syntax highlighting
- Pokemon FireRed biking sprite avatar
- Terminal aesthetic with animated background
